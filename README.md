# POS + NER Streamlit App

This app predicts **Part-of-Speech (POS)** and **Named Entity Recognition (NER)** tags from input text using a custom BiLSTM TensorFlow model.

## Features
- Joint POS + NER prediction
- Highlighted NER visualization
- POS and NER tables
- saving in csv files

## Usage
1. Install dependencies:
```bash
pip install -r requirements.txt
```
2. Run locally:
```
    streamlit run app.py
```
